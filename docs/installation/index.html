    Installation¶

    Want to install NONEVIM, here's how

    Requirements¶

    Before installing NONEVIM you need to have these things in order to install it

    System¶

    Program/Tool Purpose
    Git Cloning the repositery
    GCC Compiling native modules
    CURL Downloading extra resources (if needed)
    make Building plugin dependencies

    Runtime (In Neovim)¶

    Program/Tool Purpose
    ripgrep For live_grep and grep_sring for telescope
    fd Finder (for telescope)
    tree-sitter-cli Treecitter CLI Interface (for nvim-treesitter)

    LSP¶

    LSP Purpose
    lua_ls LUA LSP
    pyright Python LSP
    clangd C/C++ LSP
    ts_ls JS/TS LSP
    html HTML LSP
    cssls CSS LSP
    marksman Markdown LSP
    emmet_ls Emmet Extension-like on VSCode imported to Neovim

    Formatter¶

    Formatter Purpose
    stylua Lua Formatter
    pretter JS/TS, HTML, CSS Formatter
    clang-format C/C++ Formatter
    autopep8 Python Formatter

    Backup your exisiting configuration¶

    If you have an exisiting configuration, it's recommended to create a copy of it or move it to a safe place

    Linux/MacOS¶

    1
    mv ~/.config/nvim/ ~/.config/nvim-backup

    Windows (Powershell)¶

    1
    Move-Item $env:LOCALAPPDATA\nvim $env:LOCALAPPDATA\nvim_backup

    Installation¶

    Now, after you installed all the requirements and backed up your exisiting configuration, let's install NONEVIM

    1
2
    # Clone the repositery
git clone --depth=1 https://github.com/nothingfr87/NONEVIM.git ~/.config/nvim

    Linux/MacOS¶

    1
2
3
    # Remove unecessary files
cd ~/.config/nvim
rm -rf .git/ README.md screenshot* docs/

    Windows (Powershell)¶

    1
    git clone --depth 1 https://github.com/nothingfr87/NONEVIM.git $env:LOCALAPPDATA\nvim
