    Keymaps¶

    This document contains all keybindings available in NONEVIM.

    Plugin Keymaps¶

    File Explorer (NeoTree)¶

    Sidebar Management¶

    Keymap Action
    Ctrl + b Toggle NeoTree
    Space + e Focus NeoTree

    Fuzzy Finder (Telescope)¶

    Important

    The Telescope plugin requires fd to be installed on your system

    File Search¶

    Keymap Action
    Ctrl + f Find Files
    Ctrl + g Live Grep / Search Text

    Integrated Terminal (ToggleTerm)¶

    Terminal Management¶

    Keymap Action
    Ctrl + t Toggle Terminal

    Terminal Mode¶

    Keymap Action
    Esc Exit Terminal Mode

    Render Markdown¶

    Toggling¶

    Keymap Action
    Space + m + d Toggle Render Markdown

    Live Server¶

    Toggling¶

    Keymap Action
    Space + r Toggle Live Server

    Code Runner¶

    Execution¶

    Keymap Action
    Ctrl + r Run Current File

    UFO (Folding)¶

    Fold Management¶

    Keymap Action
    z + Shift + r Open All Folds
    z + Shift + m Close All Folds

    Bufferline¶

    Buffer Navigation¶

    Keymap Action
    Tab Next Buffer
    Shift + Tab Previous Buffer

    Buffer Selection¶

    Keymap Action
    Space + p Pick Buffer

    Buffer Closing¶

    Keymap Action
    Space + w Pick Buffer To Close

    Git Signs¶

    Git Visualization¶

    Keymap Action
    Space + g + w Toggle Git Diff Highlighting
    Space + g + d Toggle Deleted Lines
    Space + g + b Toggle Current Line Blame

    Core NONEVIM Keymaps¶

    Window Management¶

    Window Splitting¶

    Keymap Action
    Space + t + h Split Window Horizontally
    Space + t + v Split Window Vertically

    Window Closing¶

    Keymap Action
    Ctrl + q Force Close Current Window

    Window Navigation (Built-in)¶

    Keymap Action
    Space + h Move To Left Window
    Space + j Move To Lower Window
    Space + k Move To Upper Window
    Space + l Move To Right Window

    Window Resizing (Built-in)¶

    Keymap Action
    Space + > Increase Width
    Space + < Decrease Width
    Space + + Increase Height
    Space + - Decrease Height

    Language Server Protocol (LSP)¶

    Code Intelligence¶

    Keymap Action
    Space + h Toggle Inlay Hints
    Space + f Format Current File

    Folding¶

    Manual Folding¶

    Keymap Action
    z + f Create Fold Selection

    Built-in Neovim Keymaps¶

    Navigation¶

    File Navigation¶

    Keymap Action
    gg Go To Top Of File
    Shift + g Go To Bottom

    Scrolling¶

    Keymap Action
    Ctrl + d Scroll Half Page Down
    Ctrl + u Scroll Half Page Up
    Ctrl + f Scroll Full Page Down
    Ctrl + b Scroll Full Page Up

    Searching¶

    Keymap Action
    / Search Forward
    ? Search Backward
    n Next Match
    Shift + n Previous Match

    Editing¶

    Undo & Redo¶

    Keymap Action
    u Undo
    Ctrl + r Redo

    Copy & Paste¶

    Keymap Action
    yy Copy Line
    dd Delete Line
    p Paste After Cursor
    Shift + p Paste Before Cursor
