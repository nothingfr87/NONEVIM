    Plugins¶

    This page documents every plugin included in NONEVIM and explains its role within the distribution.

    Development¶

    Language Server Protocol (LSP)¶

    These plugins provide IDE-like features such as diagnostics, code navigation, hover documentation, code actions, and intelligent code completion.

    Plugin Purpose
    nvim-lspconfig Configure and manage Language Servers
    nvim-cmp Completion Engine for LSP, snippets, and buffer sources

    Syntax Parsing & Highlighting¶

    These plugins improve syntax awareness and provide more accurate highlighting than traditional regex-based syntax highlighting.

    Plugin Purpose
    nvim-treesitter Modern Syntax Parsing and Highlighting

    Code Formatting¶

    These plugins ensure code follows consistent formatting rules.

    Plugin Purpose
    conform.nvim Format Files Using External Formatters

    User Interface¶

    Colorscheme¶

    These plugins control the overall visual appearance of NONEVIM.

    Plugin Purpose
    catppuccin Main Colorscheme for NONEVIM

    Statusline¶

    Displays editor information such as mode, file name, Git branch, diagnostics, and cursor position.

    Plugin Purpose
    lualine.nvim Modern Statusline

    Buffer & Tab Management¶

    These plugins improve navigation between open files.

    Plugin Purpose
    bufferline.nvim Display Open Buffers as Tabs

    File Explorer¶

    These plugins provide a graphical file tree for navigating projects.

    Plugin Purpose
    neo-tree.nvim Modern File Explorer

    Icons¶

    These plugins provide icons used by other UI plugins.

    Plugin Purpose
    nvim-web-devicons File Type Icons

    Indentation Visualization¶

    These plugins help visualize code structure.

    Plugin Purpose
    mini.indentscope Animated Indentation Guides

    Comment Enhancement¶

    Improve readability of TODOs, warnings, notes, and other comments.

    Plugin Purpose
    better-comments.nvim Highlight and Stylize Comments

    Markdown Experience¶

    These plugins improve writing and reading Markdown documents.

    Plugin Purpose
    render-markdown.nvim Real-Time Markdown Rendering

    Color Visualization¶

    These plugins display colors directly inside code.

    Plugin Purpose
    nvim-highlight-colors Preview HEX, RGB, RGBA, HSL, and Named Colors

    Navigation & Search¶

    Fuzzy Finding¶

    Fast searching for files, text, buffers, and symbols.

    Plugin Purpose
    telescope.nvim Fuzzy Finder and Search Tool

    Keybinding Discovery¶

    Provides interactive keybinding hints.

    Plugin Purpose
    which-key.nvim Display Available Keybindings

    Terminal Integration¶

    Embedded Terminal¶

    Allows terminal sessions to run inside Neovim.

    Plugin Purpose
    toggleterm.nvim Integrated Floating Terminal

    Version Control (Git)¶

    Git Decorations¶

    Display Git information directly inside the editor.

    Plugin Purpose
    gitsigns.nvim Show Git Changes, Deleted Lines, and Blame Information

    Code Execution¶

    Running Programs¶

    Execute source files without leaving Neovim.

    Plugin Purpose
    code_runner.nvim Run Programs and Scripts

    Web Development¶

    Development Servers¶

    Utilities specifically useful for frontend development.

    Plugin Purpose
    live-server.nvim Launch Local Development Server with Live Reload

    Code Folding¶

    Folding Enhancements¶

    Modernize Neovim's folding system.

    Plugin Purpose
    nvim-ufo Enhanced Fold Rendering and Management

    Dependency Libraries¶

    These plugins are not directly interacted with by the user but are required by other plugins.

    Core Dependencies¶

    Plugin Purpose
    plenary.nvim Utility Functions Used by Many Plugins
    nui.nvim UI Components for Modern Neovim Plugins

    Plugin Philosophy¶

    NONEVIM aims to provide:

    • Modern IDE-like development features
    • Fast navigation and searching
    • Clean and minimal user interface
    • Git integration
    • Built-in terminal workflows
    • Enhanced code folding
    • Web development utilities
    • Sensible defaults with minimal configuration required

    This plugin selection is designed to keep NONEVIM lightweight while still providing the features expected from a modern development environment.
