Functionality / Productivity Plugins

These plugins enhance workflow efficiency, terminal usage, searching, and code execution.

toggleterm.nvim

Terminal manager inside Neovim.

Purpose

Provides floating, horizontal, or vertical terminal windows.

Features

  • Toggle terminal with keybindings
  • Multiple terminal instances
  • Floating terminal mode

telescope.nvim

Fuzzy finder framework.

Purpose

Powerful searching interface for files, buffers, and content.

Dependencies

  • plenary.nvim

Features

  • File picker
  • Live grep
  • Buffer search
  • Extensible pickers

which-key.nvim

Keybinding helper UI.

Purpose

Displays available keybindings after partial input.

Features

  • Interactive keybinding hints
  • Grouped shortcuts
  • Lazy-loaded on key press

code_runner.nvim

Code execution plugin.

Purpose

Run code snippets or files directly from Neovim.

Features

  • Multi-language support
  • Quick execution of scripts
  • Custom command mapping
