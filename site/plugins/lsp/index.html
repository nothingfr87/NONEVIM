LSP & Development Plugins

This section covers Neovim plugins responsible for language intelligence, code completion, formatting, and syntax parsing.

neovim/nvim-lspconfig

Official LSP configuration layer for Neovim.

Purpose

Provides prebuilt configurations for Language Server Protocol (LSP) servers.

Features

  • Easy setup for language servers
  • Standardized configuration interface
  • Works with nvim-lsp-installer or manual LSP setup

Notes

  • Core dependency for any LSP-based workflow

nvim-treesitter/nvim-treesitter

Advanced syntax highlighting and code parsing engine.

Purpose

Uses Tree-sitter to provide precise syntax parsing instead of regex-based highlighting.

Features

  • Fast and incremental parsing
  • Better highlighting accuracy
  • Code folding and structure awareness

Build

:TSUpdate

hrsh7th/nvim-cmp

Completion engine for Neovim.

Purpose

Provides an extensible autocompletion framework.

Dependencies

  • cmp-nvim-lsp
  • cmp-buffer
  • cmp-path
  • cmp-cmdline
  • cmp-nvim-lsp-signature-help
  • lspkind.nvim

Features

  • LSP-based completion
  • Buffer and filesystem completion
  • Command-line completion
  • Snippet integration support (external)

stevearc/conform.nvim

Lightweight formatting manager.

Purpose

Handles code formatting using external formatters.

Features

  • Declarative formatter configuration
  • Supports multiple formatters per filetype
  • Async formatting support
