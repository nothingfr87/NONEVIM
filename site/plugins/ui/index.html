UI / Appearance Plugins

Plugins that enhance Neovim's visual experience and interface usability.

catppuccin/nvim

Color scheme plugin.

Purpose

Provides a modern pastel-themed color palette.

Features

  • Multiple flavor variants
  • Highly customizable UI highlights
  • Supports Lualine, Treesitter, Telescope

askfiy/visual_studio_code

VSCode-inspired theme.

Purpose

Emulates Visual Studio Code appearance.

nvim-mini/mini.indentscope

Indentation visualization.

Purpose

Highlights indentation scope for better code structure readability.

Features

  • Animated indent guides
  • Minimal overhead

Djancyp/better-comments.nvim

Enhanced comment highlighting.

Purpose

Improves readability of TODO, FIXME, and NOTE comments.

MeanderingProgrammer/render-markdown.nvim

Markdown rendering inside Neovim.

Purpose

Improves markdown readability with inline rendering.

nvim-tree/nvim-web-devicons

File icons for Neovim UI plugins.

Purpose

Adds file-type icons across UI components.

nvim-lualine/lualine.nvim

Statusline plugin.

Purpose

Customizable and fast statusline replacement.

Features

  • Theme support
  • Git integration
  • Diagnostics display

norcalli/nvim-colorizer.lua

Color preview plugin.

Purpose

Displays actual colors for hex/RGB values in code.

nvim-neo-tree/neo-tree.nvim

File explorer UI.

Purpose

Modern tree-based file manager.

Dependencies

  • plenary.nvim
  • nui.nvim
  • nvim-web-devicons

Features

  • Git integration
  • Floating and docked modes
  • Fast navigation

akinsho/bufferline.nvim

Buffer tab UI.

Purpose

Shows open buffers in a tab-like interface.

Features

  • Buffer grouping
  • Icons support
  • Integration with devicons
